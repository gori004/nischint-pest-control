# Nischint Pest Control - Ratlam
A high-performance, professional business website built for an organic pest management service.

## Tech Stack
* **React 19** + **TypeScript**
* **Tailwind CSS v4** (Modern Styling)
* **Framer Motion** (Premium Animations)
* **Lucide React** (Iconography)

## Key Features
* 🌐 **Multi-language Support:** English & Hindi toggle.
* 🌓 **Dynamic Theming:** Smooth Dark and Light mode.
* 📸 **Result Showcase:** Before/After image comparison.
* 📱 **Mobile First:** Fully responsive for all devices.
* ⚡ **Optimized Performance:** Fast loading with Vite.
